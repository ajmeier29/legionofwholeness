# Legion Of Wholeness Backend

This is a blog website that is utilizing the MERN stack.

Technologies used:
- Node.js
- Express.js
- Firestore


## Dependencies

```
├── axios@1.7.2
├── cors@2.8.5
├── dotenv@16.4.5
├── express@4.19.2
├── firebase-admin@12.2.0
├── firebase@10.12.2
└── nodemon@3.1.4
```
